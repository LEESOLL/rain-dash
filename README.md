# Rain Dash

블래스트(VLAST) 서비스개발자 사전과제 — '속도' 키워드 웹게임.

비 오는 거리를 안 젖고 도착해야 하는 가로형 픽셀아트 러너. 핵심 메커닉은 *"내가 움직일 때만 비가 내린다"* — 시간 결합.

## 개발

```bash
npm install
npm run dev
```

http://localhost:3000

## 기술 스택

- TypeScript + Next.js (App Router)
- HTML Canvas 2D (no engine)
- Tailwind CSS 4
- Vercel 배포 예정
